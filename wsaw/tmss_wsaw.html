TMSS WireSaw Prediction

About this project

This project is aimed to detect any anomly situation before B/S (Wire breakage or Machine stop unexpectedly)

After you select dates and click 'Show Result', you would see the similar chart. Here you should know what the label and what is that chart legends: Chart result tile contains tool name, lot ID and the pre-alarm mechanism we use in this model. In above example, we choose "WSAWB01", on 2020-06-07 (We check the move out time for this lot).

Next, on the right hand side you will see a bunch of legends:

  • Predict Chart - This line shows the predict line from out predict model
  • 3 Weeks Average - This line show the average predict results for previous three weeks
  • 90% - This line show the 90% binomial distrubution upper line
  • 95% - This line show the 95% binomial distrubution upper line
  • 99% - This line show the 99% binomial distrubution upper line
  • Actual B/S - This line will exist when there is a real B/S situation
  • TMSS PI-Alarm - This line will exist when Predict line is above 99% line

As for the NCU result, you will see the similar table:

It is quite simple to understand, we classify the predict results to 5 different situation:

  • Class0 and Class1 - This indicates no any predict abnormality
  • Class2 - This still indicates no any predict anomly, but risk higher than Class1
  • Class3 - This indicates a normal, keep raising might triger predict pre-alarm
  • Class4 and Class5 - This indicates dangerous predict results, might need to check the tool status ot take actions
