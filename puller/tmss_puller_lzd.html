TMSS Puller LZD Detection

About this project

This project of Auto LZD Detection (auto-LZD) is using machine vision technology to detect LZD of the growing silicon ingot in real time operation. Currently, the system can automatically monitor ZD line of crystal growth in body stage, alert in-site engineer for LZD happened, show real time status on a specific website, record the result of detection into historical log file which include date, time, tool(puller ID), class, Lot ID(Ingot), number of LZD and crystal length, ... etc. It can also calculate the ZD line tilt angle. The others will be developed in the future.
